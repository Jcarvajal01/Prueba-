# UAV IEEE 802.15.4 Data Collection

This repository contains the source code and scripts used to collect data between an Unmanned Aerial Vehicle (UAV) and IEEE 802.15.4-enabled wireless sensor nodes.

## Project Overview

The project implements a wireless data collection system where a UAV communicates with sensor nodes using IEEE 802.15.4 technology. The system includes embedded code for the controller, transmitter node, and wireless node, as well as a Python application for data reception, visualization, and report generation.

## Repository Structure

- `Python-Application/`: Python scripts executed in PyCharm.
- `Microchip-Studio/`: Embedded codes executed in Microchip Studio.
- `Dataset/`: Collected experimental data.
- `Documentation/`: Technical documentation and reports.

## Development Environment

All codes starting with `NODO` run in Microchip Studio, and the Python code runs in PyCharm.

## Main Components

1. UAV communication controller
2. IEEE 802.15.4 transmitter node
3. IEEE 802.15.4 wireless sensor node
4. Python-based graphical interface for data collection

## Technologies Used

- Python
- PyQt5
- Serial communication
- IEEE 802.15.4
- Microchip Studio
- PyCharm

## Purpose

This repository supports experimental data collection in UAV-assisted wireless sensor networks with linear topology.
